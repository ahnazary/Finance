update_valid_tickers_table()[source]

Method to update the valid_tickers table in postgres Gets all the tickers from the tickers_list table and checks if they are valid

validity check:

  • ticker must have market cap

  • ticker must have total revenue

  • ticker must have free cash flow

  • ticker must have total assets

  • ticker must have currency code
